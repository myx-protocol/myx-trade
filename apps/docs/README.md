# MYX Trade SDK Documentation

This is the official documentation website for MYX Trade SDK, built with Docusaurus.

## Documentation Structure

- **intro.md** - SDK introduction and quick start guide
- **subscription.md** - Detailed documentation for Subscription module, including WebSocket connection, real-time data subscription and other features
- **SDK_INTEGRATION_GUIDE.md** - Complete SDK integration guide with usage examples for all modules

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run start

# Build production version
npm run build

# Preview build results
npm run serve
```

## Features

- 📚 Complete SDK documentation
- 🔍 Search functionality
- 📱 Responsive design
- 🌙 Dark mode support
- 🔗 Multi-language support configuration (configured but currently using English only)
