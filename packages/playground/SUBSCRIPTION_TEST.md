# SDK Subscription 测试页面

这是一个完整的SDK订阅功能测试页面，包含实时行情数据和K线图表功能。

## 功能特性

### 1. 实时行情订阅
- **单个行情订阅**: 可以订阅特定Pool的实时价格数据
- **批量订阅**: 一键订阅所有可用的Pool行情
- **实时更新**: 价格、成交量、涨跌幅等数据实时更新
- **连接状态**: 显示WebSocket连接状态

### 2. 行情数据展示
- **价格信息**: 当前价格、最高价、最低价
- **交易数据**: 成交量、成交额
- **涨跌显示**: 价格变化和涨跌方向（红绿显示）
- **时间戳**: 数据更新时间

### 3. K线图表功能
- **多时间周期**: 支持1分钟、5分钟、15分钟、30分钟、1小时、4小时、1天
- **实时K线**: 接收实时K线数据更新
- **数据展示**: 显示OHLCV数据
- **TradingView集成**: 为TradingView图表预留接口

## 使用方法

### 1. 启动应用
```bash
cd packages/playground
pnpm dev
```

### 2. 连接钱包
- 确保已安装MetaMask或其他Web3钱包
- 点击"连接钱包"按钮
- 选择Arbitrum测试网

### 3. 订阅行情
1. **查看可用Pool**: 在"交易池列表"中查看所有可用的交易池
2. **订阅单个行情**: 点击"订阅行情"按钮订阅特定Pool
3. **批量订阅**: 点击"订阅所有行情"按钮订阅所有Pool
4. **查看实时数据**: 在"实时行情数据"表格中查看实时更新的价格数据

### 4. 查看K线图
1. **选择Pool**: 在行情数据表格中点击"查看K线"按钮
2. **选择时间周期**: 使用下拉菜单选择K线时间周期
3. **订阅K线数据**: 点击"订阅"按钮开始接收K线数据
4. **查看数据**: 在图表区域查看K线数据和原始数据

## 技术实现

### 组件结构
```
SubscriptionPage
├── 控制面板 (连接状态、批量操作)
├── 交易池列表 (显示可用Pool)
├── 实时行情数据 (价格表格)
└── K线图表 (KlineChart组件)
    ├── 时间周期选择
    ├── 订阅控制
    ├── 图表占位符
    └── 原始数据显示
```

### 数据流
1. **WebSocket连接**: 通过SDK建立WebSocket连接
2. **订阅管理**: 管理多个订阅，避免重复订阅
3. **数据转换**: 将原始数据转换为UI友好的格式
4. **实时更新**: 使用React状态管理实时更新UI

### 类型定义
- `TickerData`: 行情数据结构
- `KlineDataPoint`: K线数据点结构
- `TickersDataResponse`: WebSocket行情响应
- `KlineDataResponse`: WebSocket K线响应

## 注意事项

1. **网络要求**: 需要连接到Arbitrum测试网
2. **钱包连接**: 需要Web3钱包支持
3. **数据格式**: K线数据已转换为TradingView兼容格式
4. **错误处理**: 包含连接错误和数据解析错误处理

## 扩展功能

### TradingView集成
当前K线组件已预留TradingView集成接口，可以：
1. 安装TradingView图表库
2. 实现`datafeed`接口
3. 配置图表选项
4. 处理实时数据更新

### 更多图表类型
可以扩展支持：
- 深度图
- 成交量图
- 技术指标图
- 多时间周期对比

## 故障排除

1. **连接失败**: 检查网络连接和钱包状态
2. **数据不更新**: 检查WebSocket连接状态
3. **订阅失败**: 检查Pool ID是否正确
4. **图表不显示**: 检查K线数据格式是否正确
